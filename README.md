`Nuxt 4` `Vue 3` `Supabase` `PostgreSQL` `Stripe` `MapLibre` `TypeScript`

### 🧠 [Odysseus](https://github.com/Scapeternam/Odysseus)
Self-hosted AI workspace — local-first, privacy-first. Chat with agents, deep research, document management, and email triage via IMAP. RAG on personal knowledge base.
`Python` `ChromaDB` `MCP` `Ollama` `FastAPI` `vLLM`

`React 18` `TypeScript` `Stripe Connect` `Supabase` `Tailwind` `Vercel`
